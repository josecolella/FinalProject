Data Visualization as a Service

Nowadays data is being produced at a faster rate than ever before. Data is great but it needs to be processed and analyzed to provide meaningful insight. As humans, we can infer much more from a visual representation of data than the processed data.

By providing a workspace with basic visualization models as well as the ability to import data sets and export to various formats, the user is empowered to create and share models representative of the data sets.

JavaScript

Using the amazing visualization libraries created in JavaScript, many of the visualization models that are seen use the SVG to allow the user to manipulate the model as he or she pleases

Python

Python is a general purpose programming language that provides great tools for data analysis and manipulation. Python provides robust data analysis tools such as numpy and pandas that allow for quick processing of large data sets.

R

R is a statistics-based programming language that provides many modules for the processing and analysis of data.

Three languages coming together. Providing an apt environment.

The goal that this platform tries to accomplish is to provide the user with the tools needed to visualize data sets.
